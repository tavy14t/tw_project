Topic-Based Resource eXplorer

This project is intended to be a Web tool for collecting, retrieving and viewing information about scientific and / or technological resources (articles, books, technical reports, presentations, source code and others) of people interested in an area of Computer science - for example, algorithms, human-computer interaction, programming paradigms, security, Web, etc.

This web tool is designed as a mash-up application, using information and media from different social media applications (such as Feed, GitHub, Google Scholar, Lanyrd, Pocket, Slideshare, Vimeo, etc.), which can be available to people interested in one or more specific area of Computer Science.
